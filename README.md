# Plemiona API

## Instruction
```bash
# Clone repo
git clone git@github.com:MendelDamian/plemiona-engine.git
cd plemiona-engine

# Create venv
python3 -m venv venv
# Activate venv
source venv/bin/activate

# Install packages
pip install -r requirements

# Migrate
python manage.py migrate

# Run development server
python manage.py runserver
```
